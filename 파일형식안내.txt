  지체보상금 계산 - 파일 형식 안내
══════════════════════════════════════════════════════

【1】 input\월별잔고회전일\  ← 월말 기준 파일 (파일마다 1개월)
  파일명: 연도_월로 구분 가능하면 됨
    예: 2024_06.xlsx / 2024년6월.xlsx / 202406.xlsx

  필수 컬럼 (이름이 달라도 자동 인식):
  ┌──────────────┬──────────────────────────────┐
  │ 판매처코드   │ 거래처코드, 코드              │
  │ 판매처명     │ 거래처명, 거래처              │
  │ 채널         │ DW07, DW11 등                 │
  │ 미수금       │ 잔고, 미수잔액, 현미수금       │
  │ 회전일       │ 현회전일, 현 회전일, 순실회전일│
  └──────────────┴──────────────────────────────┘

【2】 input\기준회전일\  ← 거래처별 기준 (1개 파일)

  필수 컬럼:
  ┌─────────────────┬──────────────────────────────────────┐
  │ 판매처코드       │ 거래처코드                            │
  │ 판매처명         │ 거래처명                              │
  │ 채널             │ DW07 / DW11 (26년 이후엔 DW_씽크 등) │
  │ 기준회전일       │ 약정회전일, 표준회전일                │
  │ 지체보상금여부   │ Y/N 또는 O/X 또는 있음/없음           │
  │ 요율 (선택)      │ 없으면 1/1000 자동 적용               │
  └─────────────────┴──────────────────────────────────────┘

【3】 output\  ← 계산 결과 자동 저장
  지체보상금_계산결과_YYYYMMDD.xlsx
  - 시트1: 전체내역 (월별 거래처별 상세)
  - 시트2: 거래처별요약 (피벗: 거래처 × 월)
  - 시트3: 월별합계
